# Project goals
As we stated in the README, this project aims to achieve pure peer-to-peer network in term of data accessibility, security, and integrity.<br>

We understand that this project is far away from finish, there are a lot of things to do. So your contributions, even if just ideas are count and meaningful to us.

For future updates, the features are shown below.

- DHT (Distributed Hash Table) with Kademlia algorithm
- RPC message handling
- Cryptographic message verificator with Rust
- Gossip protocol
- Organizing the project directory structure
- TCP protocol for chat and forum data exchange

Everything beyond will be listed as the project grows.
